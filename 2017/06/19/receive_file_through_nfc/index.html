通过 NFC 分享文件之接收文件

发表于 2017-06-19 | 分类于 Content Sharing |

当 NFC 文件接收完成后，在接收设备上通知栏上会有一个包含 Action 为 ACTION_VIEW, 第一个传输的文件的 MIME 类型以及指向它的 URI的 Intent.用户点击该通知后，该 Intent 会向系统发出，要显示接收到的数据，需要响应该 Intent, 在对应的 <activity> 标签中添加 <intent-filter>元素.
