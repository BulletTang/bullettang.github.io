通过 NFC 分享文件之发送文件

发表于 2017-06-19 | 分类于 Content Sharing |

注意事项

  1. Android 4.1(API level 16)及更高才支持大文件传输.
  2. 想要传输的文件必须保存在外部储存.
  3. 传输的每个文件都必须是全局可读 world-readable,可通过 File.setReadable(true,false) 设置.
  4. 必须提供想要传输的文件的 URI .Android Beam 文件传输不能处理 FileProvider.getUriForFile 生成的内容 URI.

需求权限

  • NFC
    <uses-permission android:name="android.permission.NFC" />
  • 读取外部储存
    <uses-permission android:name="android.permission.READ_EXTERNAL_STORAGE" />

需求硬件特性

  • NFC
    <uses-feature android:name="android.hardware.nfc" android:required="true" />
