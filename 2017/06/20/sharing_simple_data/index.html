通过 intent 分享简单数据

发表于 2017-06-20 | 分类于 Content Sharing |

分享简单数据的关键是通过一个 ACTION_SEND 的 intent.

发送简单数据给另一个 APP

发送二进制数据

分享二进制数据也是通过 ACTION_SEND,但需要设置正确的 MIME 类型，以及把数据的 URI 放到 EXTRA_STEAM 中。

注意:可以使用 */* 作为 MIME 类型，但只会匹配到可以处理基础数据流的 Activity.

发送多内容

要发送多于一个的内容，可以使用 ACTION_SEND_MULTIPLE 以及 URI 列表。

如果发送的照片类型都是 JPG，则 MIME 类型可以设置成 image/jpeg ，如果是混合不同格式图片，可以设置成 image/* ，如果不只图片数据，可以设置成 */* .

更新菜单项

将菜单 item 的 actionProviderClass 属性设置为 android.widget.ShareActionProvider

设置分享的 intent

找到分享按钮对应的菜单 item ,调用 item.getActionProvider() ，得到 ShareActionProvider 实例后调用 setShareIntent().

private ShareActionProvider mShareActionProvider;
...

@Override
public boolean onCreateOptionsMenu(Menu menu) {
    // Inflate menu resource file.
    getMenuInflater().inflate(R.menu.share_menu, menu);

    // Locate MenuItem with ShareActionProvider
    MenuItem item = menu.findItem(R.id.menu_item_share);

    // Fetch and store ShareActionProvider
    mShareActionProvider = (ShareActionProvider) item.getActionProvider();

    // Return true to display menu
    return true;
}

// Call to update the share intent
private void setShareIntent(Intent shareIntent) {
    if (mShareActionProvider != null) {
        mShareActionProvider.setShareIntent(shareIntent);
    }
}
